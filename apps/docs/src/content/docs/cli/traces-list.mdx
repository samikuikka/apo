## Synopsis

```bash
apo traces list [--task <id>] [--limit <n>]
```

## Options

| Option | Default | Description |
|---|---|---|
| `--task <id>` | all | Only traces for this task. |
| `--limit <n>` | 20 | Maximum traces to list. |
