## Synopsis

```bash
apo runs list [--task <id>] [--status <s>] [--limit <n>]
```

## Options

| Option | Default | Description |
|---|---|---|
| `--task <id>` | all | Only runs of this task. |
| `--status <s>` | all | Filter by status (e.g. `completed`, `running`). |
| `--limit <n>` | backend default | Maximum runs to list. |
